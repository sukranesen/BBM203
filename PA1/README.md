# Artifact Management System

## 📌 Description
This project is a command-based simulation system that manages artifacts and researchers. It handles assignment, tracking, and reporting operations.

## 🚀 Features
- Manage artifacts and researchers
- Assign and return artifacts with validation rules
- Maintain consistency between data structures
- Process commands from an input file

## 🧠 Data Structures
- AVL Tree (for artifact management)
- Red-Black Tree (for researcher management)

## ⚙️ How to Run
```bash
g++ -std=c++11 *.cpp *.h -o program
./program < input.txt
```

## 🛠️ Technologies
- C++
- Object-Oriented Programming
